Backend
```
# go to the backend folder
cd be

# install the dependencies
yarn install

# start the server
yarn start:dev

# run tests
yarn test:e2e

# Api endpoint
curl --location 'localhost:3333/find-server'
```


Frontend
```
# go to the frontend folder
cd fe

# install the dependencies
yarn install

# start the server (http://localhost:3000)
yarn dev

# open storybook (http://localhost:6006)
yarn storybook
```
